# Ceynova Admin Dashboard

This is the standalone admin dashboard for Ceynova Digital Solutions. It has been isolated from the main website frontend to allow separate deployments and setups.

## Credentials

To log into the admin dashboard:

- **Username:** `admin`
- **Password:** `ceynova2026`

## Getting Started

### Development

To start the development server:

```bash
npm install
npm run dev
```

### Production Build

To build the project for production deployment:

```bash
npm run build
```

The compiled output will be generated inside the `dist/` directory, ready to be deployed to your hosting provider.
